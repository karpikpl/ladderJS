Babel Starter Kit

JavaScript library boilerplate, a project template for authoring and publishing JavaScript libraries built with ES6+, Babel, Browserify, BrowserSync, Mocha, Chai, Sinon.

Features

    ✓ Next generation JavaScript via Babel<br>     ✓ Publish to NPM as ES5, ES6+ and UMD<br>     ✓ Pre-configured tests with Mocha, Chai and Sinon<br>     ✓ Project documentation boilerplate (demo)<br>     ✓ Yeoman generator (generator-javascript)<br>     ✓ Cross-platform, minimum dependencies<br>
